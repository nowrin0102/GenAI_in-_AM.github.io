Project 1: Current State and Benchmarking of Generative Artificial Intelligence for Additive Manufacturing

This project aims to identify an initial set of qualitative metrics for evaluating GenAI models and propose a framework for benchmarking based on these metrics.

Project Details

This project presents a comprehensive methodology for evaluating the capabilities of various existing GenAI tools (GPT-4, GPT-3.5, Gemini (prev. BARD), Llama 2, DALLÂ·E 3, and Stable Diffusion) in addressing diverse AM-related tasks. We propose three categories of Qualitative metrics, totaling 35 metrics, namely agnostic, domain task, and problem task metrics. We also introduce a scoring matrix, a practical tool that can be used to assess the responses of different GenAI tools.
